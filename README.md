## ğŸ“– Project Description
Word Search Generator is a **console-based C application** that automatically generates word search puzzles.  
It creates a **dynamic square grid**, hides words **horizontally, vertically, and diagonally**, and lets users search for them interactively.  
Words are loaded from external text files, and guesses are validated using a dictionary. Correct words are highlighted in the grid for **visual feedback**.
